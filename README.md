# 🚀 TaskPilot AI

## Client Onboarding Agent

This project is an Agentic AI system that automates the client onboarding process using real AI (Google Gemini).

## Features
- AI-generated welcome email using Gemini API
- Automated onboarding workflow (Agent-based)
- Generates onboarding summary dynamically
- Clean and interactive UI using Streamlit

## Tech Stack
- Python
- Streamlit (Frontend UI)
- Google Gemini API (LLM)

## Purpose
To automate manual onboarding tasks, save time, and reduce human errors.

## Run Project
pip install streamlit google-generativeai  
streamlit run app.py
