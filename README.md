# TaskPilot AI 🚀

## Client Onboarding Agent

This project is an Agentic AI system that automates the client onboarding process.

## Features
- Automated welcome email generation
- Folder structure setup
- Notion & Airtable simulation
- End-to-end workflow automation

## Tech Stack
- Python
- Streamlit
- Custom Agent Logic

## Purpose
To reduce manual onboarding effort and improve efficiency using AI.
